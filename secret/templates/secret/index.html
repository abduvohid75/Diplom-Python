Это стартовая страница проекта




API имеет следующие методы:

/generate: генерирует секрет: в запросе принимает секрет{secret}, парольную фразу{passphrase}, срок хранения{time_to_live} и возвращает secret_key.

Пример тела запроса:
{ "secret": "некоторые секретные вещи", "passphrase: "мой секрет", "time_to_live": 1000 // минут, макс=100000, по умолчанию=43200(2 месяца) }

/secrets/{passphrase}>: принимает парольную фразу в запросе URL-адреса и возвращает секрет /secrets/{secret_key}>: принимает секретный ключ в запросе URL и возвращает секрет
